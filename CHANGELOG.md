# Changelog

## v1.0.0 — 2026-06-24

Rilis awal `sr-ma-ideation`.

- Alur dua fase ideasi: gali ide (mapping → gaps → kandidat) dan saring & vonis (novelty → PICO → feasibility → rekomendasi).
- Triangulasi novelty tiga sumber: PubMed, PROSPERO, Epistemonikos/Cochrane.
- Aturan keras anti-simulasi pencarian dan atribusi PubMed + DOI.
- Cakupan dipersempit murni ke ideasi (tanpa penulisan SR/MA, PRISMA, instrumen RoB, atau analisis statistik).
- Empat template tabel: PICO, studi pendukung, decision matrix, ringkasan final.
- Contoh run penuh: genetika leukemia anak.
